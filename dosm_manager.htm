The Dependent Option Set Manager is able to:
  • Create new Web Resources inside Solutions
  • Update existing Web Resources
  • Delete Web Resources
  • Update selected Forms to add/remove Libraries and Event Handlers
Although the code used in this software is fully supported, these operations are considered dangerous if done automatically (buttons executing them are in red color).
The software provides instructions if a user wants to implement manually the functionalities.
